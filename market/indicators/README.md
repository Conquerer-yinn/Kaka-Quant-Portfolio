# 行情指标目录

这个目录用于存放行情分析指标代码，以及指标相关说明。

建议做法：

1. 指标计算代码放在这里
2. 字段口径、单位定义、调整原因，也尽量放在这里
3. 如果某个指标后续会频繁调整，说明和代码最好放得更近

这样后面修改指标时，不容易忘记原来的口径约定。

## 当前已落地模块

1. `daily_basics.py`
   每日基础数据指标。
2. `sentiment_market.py`
   总市场量能、涨跌停、炸板、大回撤、最高连板。
3. `sentiment_height.py`
   全市场、主板、创业板近十日高度观察。
4. `sentiment_chinext.py`
   创业板整体温度和涨停股质量。
5. `sentiment_feedback.py`
   创业板涨停 / 炸板 / 核心股次日反馈。
6. `position_metrics.py`
   位置度量和相对中枢判断。

## 第一阶段关键口径

1. `炸板数`
   优先使用 `limit_list_d` 中 `limit=Z` 的样本数；缺失时再回退到“触板未封住”的价格口径。
2. `大回撤数`
   从日内最高到收盘回撤大于等于 7% 的个股数。
3. `创业板强势股样本`
   第一阶段以当日创业板涨停股为主。
4. `创业板核心股`
   第一阶段先用“昨日创业板涨停股中成交额最大的个股”作为跟踪对象。
5. `十日高度`
   采用收盘价相对 10 个交易日前收盘价的累计涨幅，并排除 ST 与上市未满一年的个股。
