# 项目状态摘要

Kaka_Quant 当前提供一套围绕 A 股研究工作流的轻量项目结构，主要由以下部分组成：

1. `market/`：行情任务、指标、服务与卡片视图。
2. `strategies/`：策略研究说明与脚本预留位置。
3. `backend/`：FastAPI 服务层，负责任务触发、历史数据接口与卡片接口。
4. `frontend/`：React 展示层，负责首页、历史数据页、卡片页与策略页。
5. `storage/`：Excel 数据文件与辅助工具。

项目的主要展示能力包括：
- 历史主表维护与读取。
- 后端任务触发与状态查询。
- 前端历史数据展示。
- 飞书卡片刷新与发送。

这份文件只保留当前结构与能力摘要，不记录开发时间线。
